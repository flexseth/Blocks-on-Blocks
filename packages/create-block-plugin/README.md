# Prerequisites
- Node.js runtime environment
- Local web server

# Instructions
- Initialize NPM packages - `npm init`
- Install @wordpress/scripts - `npm install @wordpress/scripts --save-dev`
- Add start and build scripts
- create `src` folder
- create `build` folder
- double check dependencies
